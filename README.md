Веб-приложение на Spring Boot, реализующее информационно-справочную систему музыкальных фестивалей с поддержкой ролевого доступа (USER, ORGANIZER, ADMIN, SUPER_ADMIN), фильтрацией и сортировкой событий, регистрацией участников и аналитикой. Интерфейс построен с использованием Thymeleaf и адаптивного CSS; данные хранятся в PostgreSQL, безопасность обеспечена через Spring Security + JWT
